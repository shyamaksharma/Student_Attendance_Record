# Student_Attendance_Record

## What is this project?

This project helps students check whether they will be able to maintain the required 75% attendance.

It can:
- calculate your current attendance
- tell if you are safe or at risk
- show how many classes you can still miss

---

## What do you need?

You just need to follow the steps given below and just make sure:
- Python is installed on your computer

---

## How to install Python (if not installed)

1. Go to https://www.python.org
2. Download Python
3. Install it (make sure to check "Add Python to PATH")
4. Download any python interpretor like Visual Studio Code or Anaconda

---

## How to run the project

### Step 1: Download the files

- Click the green **Code** button on GitHub
- Click **Download ZIP**
- Extract the folder

---

### Step 2: Open the folder

Open the folder where the files are saved.

---

### Step 3: Open python Interpretor 

Navigate to the folder where you have extracted the folders and open them using the python interpretor 

---

### Step 4: Install required library

---

### Step 5: Run the program

#### Option 1: See dataset analysis using Attendance_data_analysis.py and edit Attendance_data.csv if you want to

#### Option 2: Enter your own data using Attendance_predictor.py


---

## How to use the input tool

When you run the program, it will ask:

- total classes conducted
- classes attended
- remaining classes

Enter the values and it will show:
- your attendance %
- how many classes you can miss
- whether you will be debarred or not
